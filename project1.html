Travel Website

Description: A dynamic travel website that allows users to explore destinations, read travel blogs, book trips, and manage accounts with authentication features.

  • 📌 Destination discovery with search and filter
  • 📝 Blog section featuring tips, hacks, and destination guides
  • 🔐 JWT-based login/signup with email verification
  • 👤 User profile management (update info, logout, delete account)
  • 📧 Contact & booking system integrated with backend

Tech Stack: React.js, Node.js, Express.js, MongoDB, JWT, Bcrypt, Nodemailer

Upcoming: Google Auth with Firebase, dynamic place details, and real-time booking status updates.
