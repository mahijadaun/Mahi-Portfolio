  • Interactive design with smooth transitions and animations.

Real-Time Updates

  • Get instant updates to posts and private messages.
  • All interactions (likes, comments, messages) are updated live.

üíª Setup Steps

  5. Run the server and client: 
    npm run server 
     cd client 
     npm start
